# Discussion: Design notes

The Chinese label `频道报送数目` was chosen over `每源上限` to emphasize the channel's submission volume rather than a rigid limit. The intent is forward-looking: a future survey system (see [0002](../0002_survey_preference_system/)) will replace the hard cap with user feedback like "too much" / "too little".
