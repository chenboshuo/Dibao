# 邸报 Dibao

邸报 Dibao 是一个开源、可自托管、单用户的个人化 RSS 信息流系统。

项目目标是把个性化排序限制在用户自己订阅的 RSS / Atom 信源范围内，让用户保留对信息来源、数据和推荐逻辑的控制权。
